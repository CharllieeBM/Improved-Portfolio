About Me

Learning as I code

My love for web design started in 2001 when I was in grade 6, thanks to a website called AsianAvenue that let you customize your own homepage using pure HTML. I was hooked immediately.

From that moment, I was constantly tweaking how my page looked — experimenting with styles, and when I didn't know the code, I'd right-click other pages just to see how they did it. Long before I knew what CSS was, I was already thinking like a developer.

Fast-forward to today — I've completed a Front-End Web Developer course with SheCodes and professional-level training at EPAM, where I built a full multi-page e-commerce application using TypeScript, SASS, and vanilla JavaScript. This portfolio is a showcase of that journey, featuring projects in React.js, responsive design, HTML, and CSS. I hope you enjoy exploring it!

Featured Project

EPAM Suitcase E-Shop

My most ambitious project to date — a fully responsive, multi-page e-commerce store built during professional-level training at EPAM. No frameworks allowed: just TypeScript, SASS, and vanilla JavaScript working together to deliver a polished shopping experience.

It features a dynamic product catalog with filtering, a fully working cart with LocalStorage persistence, individual product detail pages, a contact form, and an About page — all driven by local JSON data and built to a Figma design spec.
